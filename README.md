# opensource COBOL 4j development environment (Docker)

Versions :
- CentOS: centos8
- opensource COBOL 4j: v1.0.3

In order to "Hello World" program, run the following commands in the docker container

```
cd /root/cobol_sample
cobc HELLO.cbl
java HELLO
```
